# Лабораторна робота №4  
## Імплементація інтеграції з віддаленими джерелами даних

### Тема
Student Notes System (Node.js)

### Мета роботи
Замінити використання статичних даних у застосунку на реальну роботу з базою даних.

### Опис
У даній лабораторній роботі реалізовано інтеграцію консольного застосунку з базою даних SQLite.
Дані нотаток більше не зберігаються у памʼяті програми, а записуються та зчитуються з реальної бази даних.

### Реалізовані сценарії
- додавання нотаток
- перегляд нотаток
- редагування нотаток
- видалення нотаток

### Використана база даних
SQLite

### Запуск
```bash
node index.js
