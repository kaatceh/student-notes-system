# Лабораторна робота №1

## Налаштування проекту та інструментів розробки

У цій лабораторній роботі було:

- ініціалізовано Node.js-проєкт
- обрано стиль коду
- підключено форматтер Prettier
- налаштовано лінтер ESLint
- налаштовано Git hooks (pre-commit, pre-push)

Під час коммітів перевіряється форматування, лінтер та тести.
Під час push перевіряється запуск проєкту.
