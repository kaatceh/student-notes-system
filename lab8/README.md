# Лабораторна робота №8

## Мета роботи
Навчитися знаходити вузькі місця (bottlenecks) у системі, 
виділяти їх у окремий модуль та проводити рефакторинг із застосуванням
інʼєкції залежностей.

## Виконання
- Визначено основний bottleneck: робота з нотатками (service + repository)
- Виділено модуль notes-core
- Застосовано Dependency Injection через конструктор
- Логіка роботи з БД ізольована від UI
- Сервіс підготовлений до подальшого винесення в окрему бібліотеку
