# Лабораторна робота №6
## Розгортання програмного забезпечення (CI/CD)

У цій лабораторній роботі налаштовано:

- CI за допомогою GitHub Actions
- Перевірку стилю коду (Prettier)
- Перевірку коду ESLint
- Запуск тестів
- Збірку проєкту
- Захист гілки main (merge тільки після проходження CI)

Розгортання відбувається автоматично після злиття Pull Request у гілку main.
