# Student Notes System (Node.js)

## Опис ідеї

Student Notes System — це консольний застосунок на платформі Node.js для збереження та
керування навчальними нотатками студентів.

Проєкт має навчальний характер і буде поступово розширюватися у наступних лабораторних роботах.

## Мета проєкту

- Створити застосунок для збереження нотаток
- Навчитися працювати з Node.js
- Підготувати основу для наступних лабораторних

## Основні можливості

- Додавання нотаток
- Перегляд нотаток
- Редагування нотаток
- Видалення нотаток
