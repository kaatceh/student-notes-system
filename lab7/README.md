# Лабораторна робота №7 – Перформанс

## Мета роботи

Проаналізувати продуктивність програмного забезпечення, використовуючи інструменти профілювання та навантаження.

## Використані інструменти

- Node.js
- SQLite
- Clinic.js (clinic doctor)

## Виконані дії

- Проведено профілювання продуктивності застосунку за допомогою clinic doctor
- Проаналізовано використання CPU та RAM
- Проаналізовано затримки event loop
- Проведено аналіз запитів до бази даних SQLite
- Перевірено типові сценарії використання застосунку під навантаженням

## Профілювання продуктивності

Для аналізу продуктивності використано clinic doctor.
Отримано графіки CPU, RAM та Event Loop під час типових сценаріїв
(додавання, перегляд, редагування нотаток).
Скріншоти результатів наведені у папці screenshots.

## Запуск аналізу

```bash
npx clinic doctor -- node src/index.js
```
