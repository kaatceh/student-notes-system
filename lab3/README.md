# Лабораторна робота №3

## Імплементація інтерактивного прототипу

### Тема

Student Notes System (Node.js)

### Мета роботи

Імплементувати інтерактивний прототип застосунку з використанням статичних даних без підключення до бази даних.

### Опис

Реалізовано консольний прототип для керування нотатками студентів.
Дані зберігаються у памʼяті програми.

### Реалізовані сценарії

- додавання нотаток
- перегляд нотаток
- редагування нотаток
- видалення нотаток

### Запуск

node index.js
