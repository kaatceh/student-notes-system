# Лабораторна робота №5

## Тестування програмного забезпечення

### Мета роботи

Реалізувати unit, integration та E2E тестування для застосунку Student Notes System, а також провести мутаційне тестування для оцінки ефективності тестів.

### Unit-тестування

Модуль роботи з нотатками (`notesService.js`) покрито unit-тестами, що перевіряють основні CRUD-операції.

### Інтеграційне тестування

Налаштовано інтеграційні тести з використанням реальної SQLite бази даних, що дозволяє перевірити коректність взаємодії модулів.

### E2E тестування

Реалізовано E2E тест, який охоплює повний сценарій роботи системи: ініціалізацію бази даних, додавання та отримання нотатки.

### Мутаційне тестування

Проведено мутаційне тестування за допомогою Stryker.  
Результати показали, що бізнес-логіка застосунку покрита тестами значно краще, ніж консольний інтерфейс, що є очікуваним для даного типу проєкту.

### Запуск тестів

```bash
npm test
npx stryker run
```
